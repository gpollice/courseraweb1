The <header> element contains some header information that might consist of a logo, tagline, etc. It can also contain some navigation using the <nav> elements.

All the tags described in this page are block level. They could be replaced by just <div> tags. The elements used here are for describing structure of the document to the reader. They don't, however give more functionality than the <div> block elements.
